1. 如何购买正版

官方的中国代理商列表链接

建议购买终身版。各个代理商之间的产品并没有什么差异，看个人的爱好购买。 多谢支持正版，让IDM能够走得更远。

2. 使用盗版后，购买正版使用正版序列号仍然提示虚假的序列号

首先多谢支持正版

出现的原因

因为盗版的软件在你的注册表内写入了虚假的序列号没有被删除掉，所以你现在使用正版的序列号进行注册提示虚假序列号。

解决方案

  • 下载 清理注册信息的文件 [opensource]IDM-trial-reset.zip IDM吧群共享

  • 解压并运行 idmtrialreset.exe

  • 现在 右键以管理员权限运行IDM，并使用序列号注册IDM 注册成功后，退出IDM进程。之后就可以已普通运行的方式运行IDM（普通运行方式就是不必再 右键以管理员权限运行IDM）

3. 安装过程中弹出对话框

目前来看，在安装过程中只弹出来一种对话框。

对话框提示内容：

  • 如果你将IDM安装到所选的目录，IDM将不能在保护模式下的IE内集成。

    如果需要集成到保护模式下的IE，必须安装到C盘。

    你想将目录切换到C盘么？

    是（切换到C盘） 否（保持安装到其他盘）

出现的原因：
  • 更换了IDM的默认安装目录。
解决方案：
1. 点击是，将安装目录切换回C盘。
2. 点击否，继续安装到其他盘。

4. 如何更改语言为中文

正常情况下，在中文的Windows上IDM在安装完成之后会提示需要下载中文语言包，下载安装完成，重新打开IDM即可。

不正常的情况下
  1. 下载IDM官方的语言包：链接
  2. 解压语言包到安装目录的Languages文件夹内。例如：C:\Program Files (x86)\Internet Download Manager\Languages

  3. 重新启动IDM在View->Language内选择->简体中文

5. 如何关联Firefox，chrome以及国产双核浏览器

1. 如何安装浏览器扩展

  • 1.1 chrome及国内的双核浏览器如何添加(火狐直接看最后的，IE请查看其他教程)。

    (1)打开浏览器的扩展工具页面，不要关闭。

    (2)打开C(D/E/F...):Program Files (x86)\Internet Download Manager 寻找IDMGCExt.crx

    (3)单击按住IDMGCExt.crx文件不要放，拖动到2.1步打开的扩展工具页面，然后放开，提示是否安装，安装。重启浏览器OK。

  • 1.2 firefox(火狐/FF)浏览器如何添加

    注意：
    • Firefox 53及更新的版本使用 idmmzcc3.xpi
    • Firefox 27---52 使用 idmmzcc2.xpi
    • Firefox 1.5---26 使用 idmmzcc.xpi

    (1)打开firefox

    (2)打开C(D/E/F...):Program Files (x86)\Internet Download Manager 寻找idmmzcc.xpi(参考 注意 选择）

    (3)单击按住idmmzcc3.xpi(参考 注意 选择)文件不要放，然后拖放到火狐中，放开，提示安装，安装。重启浏览器OK。

2. 如何关联其他chromium（webkit/极速）内核浏览器

  • 2.1 右击浏览器快捷方式，点击属性，点击打开文件位置，不要关闭弹出的窗口

    以chrome为例

  • 2.2 打开IDM，点击下载->选项->添加浏览器，从弹出的窗口中选择第一步中找到的可执行文件（exe文件），选中即可

    同样以chrome为例

  • 2.3 安装IDM的扩展工具，见1.1所述.

7. 如何破解使用（推荐在力所能及范围内支持正版，让好的软件走的更远）

有两种简单方式进行破解。

第一种
  1. 进入IDM吧群共享：链接
  2. 选择自己喜欢的破解版本
  3. 安装下载好的破解版本
第二种
  1. 进入IDM吧群共享：链接
  2. 在官方原版文件夹内，下载最新的官方安装包
  3. 在patch文件夹内下载 看雪破解文件
  4. 安装原版安装包，并关闭IDM
  5. 将看雪破解文件内的 dll 解压到 IDM的安装目录（压缩包内含README文档，有操作方法。）
  6. 重新打开IDM

8. IDM可以做到哪些，不能做哪些

1. IDM可以做哪些？
  • IDM可以下载普通的HTTP，FTP链接
  • IDM可以抓取某些在线视频（例如YouTube）
2. IDM不能做哪些？
  • IDM不能突破你的带宽极限，它只能不留余力的去下载
  • IDM不能下载某些加密的在线视频（大多数的教程网站）
  • IDM不能下载 磁力链接，迅雷链接，种子文件，电驴链接，旋风链接。（仅仅支持 http/https/ftp 开头的下载链接）

9. 下载路径的设置

默认IDM的下载保存路径 在C:\Users\用户名\Downloads 下

如果需要修改IDM的保存路径，打开 IDM 点击 下载-->选项-->保存至

你需要注意的是，IDM对于每一种类型都有单独的存储路径，所以你需要对每一个分类都进行修改。

更改下方的临时文件夹路径到与保存路径不同的盘，可以加速文件整合的速度。

10. 多线程下载的设置

默认IDM的下载连接数为8
你需要了解的
  • 最大连接数，并不是IDM可以单方面决定的，还需要服务器端进行配合。所以会有8个线程都达不到的情况，也会有跑满32线程的情况。
  • 线程数是一把双刃剑。更多的线程数下载可以一定程度上，但并不是绝对可以加快你的下载速度，但是一定会增加你合并文件所需的时间。
如果你需要修改最大线程数

打开 IDM 点击 下载-->选项-->连接
